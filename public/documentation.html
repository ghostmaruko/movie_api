Movie API Documentation

This API allows users to retrieve movie data, register and manage accounts, and manage favorite movies.

Endpoints

Method URL Description Request Body Response Example
GET /movies Returns a list of all movies None [ { "Title": "The Lord of the Rings", "Year": 2001, ... }, ... ]
GET /movies/:title Returns data about a single movie by title None { "Title": "The Lord of the Rings", "Year": 2001, ... }
GET /genres/:name Returns the description of a genre by name None { "Genre": "Sci-Fi", "Description": "A hacker discovers..." }
GET /directors/:name Returns director details by name None { "Name": "Christopher Nolan", "Bio": "...", "Birth": 1970 }
POST /users Registers a new user { "Username": "marco", "Password": "12345", "Email": "email@example.com" } User marco registered
PUT /users/:username Updates the user data { "Username": "marco2", "Password": "67890", "Email": "new@example.com" } User updated
POST /users/:username/movies/:movieID Adds a movie to user's favorites None Movie Inception added to marco's favorites
DELETE /users/:username/movies/:movieID Removes a movie from user's favorites None Movie removed from marco's favorites
DELETE /users/:username Deletes a user account None User marco has been removed

Notes

  • All responses are in JSON format unless otherwise specified.
  • The ImagePath field points to files in the /public/img folder.
  • For POST and PUT requests, set Content-Type to application/json.
  • Movie fields include: Title, Description, Genre, Director, Year, Actors, ImagePath, Featured.
